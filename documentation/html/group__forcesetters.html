Detailed Description

For handling Compliance/PID settings for fine control Note: PID is only for MX28, MX64 and MX106 - where pidCapable(model) == true Note: Compliance Slope/Margin is only for AX12 and RX28 - where complianceSettingCapable(mode) == true
